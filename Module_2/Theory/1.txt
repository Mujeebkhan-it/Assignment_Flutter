1. Explain the fundamental data types in Dart (int, double, String, List, Map, etc.) and their uses.

-   Data types :-- There are mainly 4 data types in dart 

1. Int - To store numbers
2. Double - To store decimal point numbers
3. String - To store String, Words, Letters
4. Boolean - To store True or False value
5. List - To store multiple values in single variables
6. Map - To store multiple values in single location and in key - value pair
